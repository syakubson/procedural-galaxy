# ✦ Galaxy — процедурная анимированная вселенная

Интерактивная спиральная галактика в браузере в духе заставки Spore: вращающийся
звёздный диск, светящееся ядро, газовая туманность и маленькие разноцветные
«солнца» солнечных систем. Всё **процедурно генерируется из сида** и настраивается
вживую панелью справа.

Главная идея под капотом — **производительность для слабых ПК**: все звёзды
рисуются за один вызов отрисовки (`THREE.Points`), а вращение, мерцание и
перспективный размер считаются **в вершинном шейдере на GPU**. На каждом кадре CPU
обновляет ровно одно число (время) — он не перебирает десятки тысяч звёзд.

---

## Запуск

ES-модули требуют HTTP (не `file://`). Поднимите любой статический сервер из папки
проекта:

```bash
cd galaxy
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

Альтернативы: `npx serve`, расширение Live Server в VS Code — что угодно, что
раздаёт статику.

> Three.js и lil-gui лежат локально в `vendor/` — интернет для работы не нужен.

## Управление

| Действие | Что делает |
| --- | --- |
| **Колесо мыши / пинч** | приблизиться к центру галактики и отдалиться |
| **Зажать ЛКМ / тащить** | повернуть камеру вокруг центра |
| **Навести на цветное кольцо** | имя и статус исследуемой системы |
| **Клик по кольцу** | варп в систему: звезда + планеты + описание |
| **«← Назад» / Esc** | вернуться из системы в галактику |
| **🔊 справа внизу** | включить/выключить космический эмбиент |
| **Панель справа** | живая настройка всех параметров |

## Настройки (панель)

- **Палитра** — 5 цветовых схем (Spore, Ember, Emerald, Ice, Rose).
- **Качество** — `Низкое` (слабый ПК) / `Среднее` / `Высокое`: масштабирует число
  звёзд, bloom, антиалиасинг и предел pixel ratio.
- **Сид** + 🎲 — детерминированная генерация: один сид → одна и та же галактика.
- **Форма** — число звёзд, рукава, закрутка, толщина рукавов, разброс, размер и
  плотность ядра, толщина диска. *(меняют геометрию — пересборка при отпускании)*
- **Солнца** — количество и размер цветных солнц.
- **Движение** — скорость вращения, дифференциальное вращение (внутренние звёзды
  быстрее), мерцание, авто-вращение камеры.
- **Свет и туманность** — яркость (экспозиция тонмаппинга), размер звёзд, газовая
  туманность и её плотность.
- **Системы** — доля исследуемых систем, метки вкл/выкл, кнопка «случайная система».

## Исследуемые системы

Часть солнц (по умолчанию ~24%) помечены **цветными кольцами** — это «настоящие»
системы, генерируемые из сида. Цвет = статус: 🟢 обитаемые · 🔵 дикие · 🟠 руины
(легенда в углу). Наведи — увидишь имя; кликни — плавный **варп** в систему.

В виде системы: процедурная звезда (своего класса, с гранулами/пятнами и ореолом-
короной из лучей) и 2–7 планет на **кеплеровских орбитах** (внешние медленнее,
ω ∝ a^−1.5), с орбитальными линиями. Планеты «похожи на реальные» — шейдер по типу:

- **земного типа** — биомы (океанический · джунгли · ледяной · пустынный · мир-город),
  материки/океаны/облака, блик на воде, день/ночь-терминатор, атмосферный ободок;
- **газовый гигант** — полосы зон/поясов, шторм, **кольца** со щелью Кассини, тенью
  планеты и прозрачностью;
- **лавовая** (светящиеся жилы), **ледяная**, **каменистая**, **пустынная**;
- у части — небольшие **луны** (реалистично мелкие и медленные).

**Обитаемые миры** (только у долгоживущих звёзд F/G/K/M) бывают на разных этапах:
**племена** (тёмная ночь) → **индустриальная эпоха** (огни + спутники) → **космическая
цивилизация** (яркие мегаполисы, орбитальная станция-кольцо, **колонии** на соседних
планетах и **корабли-огоньки**, снующие между мирами).

**Описание системы** богатое: имя, тип и нрав звезды, **возраст** (млрд лет),
**история** формирования, **ресурсы** и назначение, атмосферный обзор-лор, а для
обитаемых — отдельный блок про **расу** (кто, на каком этапе и почему). У каждой
планеты — мини-иконка-сфера.

**Вовлечение:** на hover — краткая инфо (звезда · планет · этап) + тизер «нажмите,
чтобы исследовать →» (полная история только по клику), счётчик **«Исследовано N / M»**,
непосещённые метки ярче пульсируют, а в панели каждой системы — **«💡 факт»** из
астрофизики.

### Особые объекты и детали

- **Чёрная дыра в центре галактики** (Стрелец A*) — отдельный кликабельный объект:
  чёрный горизонт, аккреционный диск (температурный градиент + кеплеровская
  закрутка + доплеровское усиление), фотонное кольцо. С описанием и историей.
- **Система «Интерстеллар»** — Гаргантюа (диск, «оборачивающийся» над и под дырой
  за счёт линзирования) и вращающаяся станция-кольцо **«Эндюранс»**, летящая по
  орбите вокруг неё.
- **Бинарные звёзды** — часть систем с двумя солнцами (близкая пара), планеты по
  циркумбинарной орбите.
- **Этапы цивилизаций**: племена (тёмная ночь) → индустрия (огни + спутники) →
  космос (мегаполисы, станция-кольцо, **колонии** на соседних мирах, **корабли**,
  что вылетают из одной планеты и прилетают в другую).
- **Руины бывают разными**: вымершие, **работающие на роботах** (холодные огни ещё
  горят) и **с кратером от катастрофы**.
- **Орбиты планет** убраны ради чистоты — вместо них у каждой планеты еле заметный
  цветной **хвост движения**.
- **Плавный вход**: при клике система появляется отдалённой и камера **доезжает
  зумом** до обзора (без резкого чёрного кадра).
- **Космический эмбиент** — спокойная музыка, сгенерированная процедурно через Web
  Audio API (никаких файлов и лицензий — полностью свободна для использования).

## Производительность

- **Один draw call** на весь звёздный диск (`BufferGeometry` + `THREE.Points`).
- **Вращение на GPU** в вершинном шейдере → нулевая покадровая работа CPU по звёздам.
- **Без текстур** — спрайты звёзд рисуются процедурно во фрагментном шейдере.
- **HDR + ACES-тонмаппинг вместо тяжёлого bloom**: ядро светится мягким бело-золотым
  свечением без дорогого многоступенчатого пост-эффекта (ореолы дают сами звёзды).
- **Размер точек ограничен** аппаратным лимитом GPU (`ALIASED_POINT_SIZE_RANGE`) —
  меньше overdraw в ядре, нет «обрыва» драйвера на слабых видеокартах.
- **Pixel ratio ограничен** пресетом качества; **сглаживание (MSAA 4×)** в HDR-
  конвейере включается на «Среднем»/«Высоком», на «Низком» — выключено.
- **Пауза рендера** на скрытой вкладке (экономия батареи/CPU).
- **Авто-снижение качества**: при стабильно низком FPS pixel ratio падает до 1.0
  (разово, без рывков).

Ориентир: «Среднее» (60k звёзд) комфортно идёт на интегрированной графике; «Низкое»
(24k) — для совсем слабых машин.

## Структура

```
galaxy/
├── index.html              # canvas + import map + оверлеи
├── styles.css
├── vendor/                 # локальные Three.js core/addons + lil-gui
└── src/
    ├── main.js             # рендерер, камера, OrbitControls, цикл, адаптивность
    ├── config.js           # дефолты + пресеты качества
    ├── palettes.js         # цветовые схемы
    ├── rng.js              # seeded PRNG (mulberry32) для воспроизводимости
    ├── galaxy.js           # генерация звёздного диска + балджа
    ├── suns.js             # слой цветных «солнц»
    ├── background.js       # дальнее звёздное поле + газовая туманность
    ├── postfx.js           # HDR-конвейер + ACES-тонмаппинг (EffectComposer/OutputPass)
    ├── gui.js              # панель lil-gui
    ├── systems/            # исследуемые системы
    │   ├── markers.js      # метки реальных систем в галактике (hover/клик/raycast)
    │   ├── systemData.js   # процедурная генерация системы из сида
    │   ├── lore.js         # атмосферные русские описания
    │   ├── planet.js       # планета: меш + шейдер + кольца + луны + орбита
    │   └── systemView.js   # сцена системы: звезда + планеты + орбиты, enter/exit
    ├── ui/
    │   └── hud.js          # панель лора, подсказка, легенда, затемнение-переход
    └── shaders/
        ├── starShader.js          # вращение/мерцание/размер звёзд галактики (GPU)
        ├── sunShader.js           # солнца с пульсацией
        ├── nebulaShader.js        # газовый диск (fbm-шум + эхо рукавов)
        ├── planetShader.js        # поверхность планет по типу (fbm, биомы, лава, газ)
        └── starSurfaceShader.js   # звезда системы: гранулы + потемнение к краю + корона
```
